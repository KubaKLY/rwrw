Projekt webowej aplikacji VR
Link: https://piotrprzybylski505.github.io/rwr/

Opis:
Prosta scena z kilkoma obiektami, po której użytkownik może się poruszać.

Sterowanie:
- Teleportacja - celowanie za pomocą prawego kontrolera, teleportacja za pomocą triggera
- Płynny ruch - joystick lewego kontrolera
- Obrót (Snap Turn) - joystick prawego kontrolera
